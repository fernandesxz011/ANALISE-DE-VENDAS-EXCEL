# Análise de Vendas - Excel

## Objetivo
Projeto desenvolvido para praticar análise de dados utilizando Microsoft Excel.

## Ferramentas Utilizadas
- Microsoft Excel
- Tabelas
- Fórmulas
- Filtros
- Análise de Dados

## O que foi analisado
- Dados de vendas
- Valores vendidos
- Quantidades
- Informações dos produtos

## Aprendizados
Durante este projeto foram praticados conceitos de organização, tratamento e análise de dados utilizando Excel.
